FLOW TIMESHEET BRICKMMO

This is feature Timesheet of BrickMMO using PHP, HTML, CSS.

The imesheet helps students track progress, access student information, view task details, and total coding hours for students to allocate hours to tasks and manage project entries

The Timesheet feature helps admin to monitor tasks' progress and support to students if necessary.

The Calendar for students choosing the day they want to implement their tasks

After choosing the date for a tasks, the student will add below information(date, hours, decription) and select the task

After adding information and the task, the student will see a list of all tasks on their screen

In admin page, the admin can track all student information and their timesheet, they can choose the timesheet of any student.

The admin can see the calendar with total time that the student inputed

The admin can see the list of tasks that the student is doing
